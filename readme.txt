ABEngine is an API that supports other plugins to create and manage A/B tests.

== Description ==

ABEngine is part of a suite of tools for news publishers, built by [MavEngine](https://mavengine.com), a division of [Daily Maverick](https://dailymaverick.co.za). 

It allows other plugins to create and manage A/B tests.
